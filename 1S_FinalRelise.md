Освобождение ресурсов (только для 1С 8.1) 
=========================================

После окончания работы с системой Диадок, необходимо освободить
используемые ресурсы.

Для этого необходимо вызвать метод
[FinalRelease](1S_DiadocObj_FinalRelease) объекта интерфейса Диадок.

Данная операция необходима только для платформы 1С 8.1.
