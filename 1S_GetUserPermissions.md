GetUserPermissions 
==================

Метод объекта [Organization](Организация)

Синтаксис
GetUserPermissions()

Возвращаемое значение
Объект типа [UserPermissions](1S_UserPermissions_Desc)

Описание
Возвращает описание прав пользователя, в контексте которого произошла
авторизация, для данной организации.
