===================================

Объектная модель данных :doc:`Invoice <Invoice>`.

Модель позволяет работать с документом исправленный корректировочный
Счет-Фактура, как со структированным объектом.

-  Torg12SellerContent содержание титула исполнителя документов типа
   «акт о выполнении работ в формате ФНС».
