Метод объекта `Organization <Организация>`__

Синтаксис
~~~~~~~~~

GetCounteragentById(\\)

Параметры
~~~~~~~~~

-  \\ (Строка, обязательный) - идентификатор контрагента

Возвращаемое значение
~~~~~~~~~~~~~~~~~~~~~

Объект `Counteragent <Контрагент>`__.

Описание
~~~~~~~~

Возвращает контрагента, по указанному идентификатору.
