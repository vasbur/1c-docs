Метод объекта `DocumentEvent <DocumentEvent>`__.

Синтаксис
~~~~~~~~~

GetFileContent(\\)

Параметры
~~~~~~~~~

-  \\ (строка, обязательный) - путь для сохранения содержимого
   документа

Возвращаемое значение
~~~~~~~~~~~~~~~~~~~~~

Строка. Исходное имя файла документа.

Описание
~~~~~~~~

Сохраняет содержимое документа на диск.
