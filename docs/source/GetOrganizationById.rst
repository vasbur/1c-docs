Метод объекта `DiadocConnection <Соединение>`__.

Синтаксис
~~~~~~~~~

GetOrganizationById(\\)

Параметры
~~~~~~~~~

-  \\ (Строка, обязательный) - идентификатор организации в
   Диадоке

Возвращаемое значение
~~~~~~~~~~~~~~~~~~~~~

Объект `Organization <Организация>`__.

Описание
~~~~~~~~

| Возвращает организацию в Диадоке с идентификатором BoxId в случае,
  если
| пользователь имеет к нему доступ.
