Метод объекта `Organization <Организация>`__.

Синтаксис
~~~~~~~~~

SendDraftAsync(\\)

Параметры
~~~~~~~~~

-  \\ (Строка, обязательный) Уникальный идентификатор черновика.

Возвращаемое значение
~~~~~~~~~~~~~~~~~~~~~

Объект `AsyncResult <AsyncResult>`__.

Описание
~~~~~~~~

| Инициирует асинхронную операцию отправки черновика по его уникальному
  идентификатору. Результатом асинхронной операции является коллекция
  `Collection <Collection>`__ объектов, производных от
| `Document <Документы>`__.
