Метод объекта `DocumentsTask <DocumentsTask>`__.

Синтаксис
~~~~~~~~~

GetDocuments()

Возвращаемое значение
~~~~~~~~~~~~~~~~~~~~~

| Коллекция `Collection <Collection>`__ объектов, производных от
| `Document <Документы>`__

Описание
~~~~~~~~

Возвращает список документов в текущем ящике по заданному фильтру.
