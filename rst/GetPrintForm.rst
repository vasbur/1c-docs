Метод объекта `Document <1S_GetPrintForm_Desc>`__

Синтаксис
~~~~~~~~~

GetPrintForm(\\, \\)

Параметры
~~~~~~~~~

-  \\ (строка, обязательный) - путь для сохранения печатной
   формы.
-  \\ (число, необязательный) - время (в секундах), в течение
   которого, ожидается формирование печатной формы.

Возвращаемое значение
~~~~~~~~~~~~~~~~~~~~~

Отсутствует

Описание
~~~~~~~~

Формирование печатной формы для данного документа.
