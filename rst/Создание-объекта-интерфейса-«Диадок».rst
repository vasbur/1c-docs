Для работы с системой Диадок, необходимо:

-  Загрузить внешнюю компоненту (процедура ЗагрузитьВнешнююКомпоненту)
-  Создать объект интерфейса Диадок.

Для создания объекта интерфейса Диадок, необходимо:

-  Создать объект расширения внешней компоненты. Создание:
   Новый(«AddIn.DiadocInvoiceAPI»)
-  У полученного объекта вызвать метод CreateObject() и получить объект
   интерфейса Диадок
