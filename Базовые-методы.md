
Базовые методы, применимые ко всем производным типам документов

-   SaveAllContent
    сохраняет все файлы, относящиеся к документу (в т.ч. электронные
    подписи), в указанную директорию
-   SaveAllContentAsync
    асинхронно сохраняет все файлы, относящиеся к документу (в т.ч.
    электронные подписи), в указанную директорию
-   GetComment
    возвращает комментарий к документу, заданный при отправке
-   Move
    перекладывет документ в указанное подразделение
-   Delete
    помечает документ как удаленный
-   SaveContent
    на основании содержания документа формирует файл документа и
    сохраняет его на диск
-   Approve
    ставит признак согласования документа
-   Disapprove
    ставит признак отказа в согласовании документа
-   SetOneSDocumentId
    устанавливает идентификатор 1С для документа
-   ReSetOneSDocumentId
    сбрасывает идентификатор 1С для документа
-   CreateResolutionRequestTask
    создает задание для отправки запроса на согласование
-   GetSenderSignature
    получает подпись отправителя, приложенную к документу
-   GetRecipientSignature
    получает подпись получателя, приложенную к документу
-   GetStructuredDataAttachment
    получает структурированные данные, описывающими те или иные
    документы, представленные в виде печатных форм
-   SendRevocationRequest
    отправляет запрос на аннулирование документа
-   AcceptRevocationRequest
    принимает запрос на аннулирование документа
-   RejectRevocationRequest
    отклоняет запрос об аннулировании документа
-   GetPrintForm
    формирование печатной формы для данного документа
