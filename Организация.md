
Объект Organization предназначен для работы с организацией на сервере
Диадок. Позволяет:

-   Работать со списком контрагентов. С помощью метода
    GetCounteragentListByStatus
    можно получить список контрагентов, с которыми установлены
    взаимоотношения в Диадоке. С помощью методов
    GetCounteragentListByInnKpp
    ,
    GetCounteragentListByInnList
    можно найти контрагента (контрагентов) по ИНН.
-   Получать список документов организации с помощью метода
    GetDocumentsTask
    .
-   Подписывать и отправлять документы через Диадок с помощью методов
    CreateSendTask
    ,
    CreateSendTaskFromFile
    .

###Свойства объекта
 Свойство | Тип значения | Доступ | Описание
 --------------------- | ----------------------------------------------------------------------------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 Id | Строка | Чтение | Идентификатор организации в Диадоке
 Name | Строка | Чтение | Наименование организации
 Inn | Строка | Чтение | ИНН организации
 Kpp | Строка | Чтение | КПП организации
 Departments | [коллекция](1S_Collection_Desc) объектов [Department](1S_Department_Desc)   Ч | тение   С | писок подразделений организации. Возвращаются подразделения, не имеющие родительских подразделений. Список дочерних подразделений можно получить, используя свойство Subdepartments объекта Department
 AuthenticateType | Строка | Чтение | Тип авторизации
 Login | Строка | Чтение | Логин, по которому произошла авторизация к данной организации
 Certificate | Объект [PersonalCertificate](1S_PersonalCertificate_Desc) | Чтение | Сертификат, по которому произошла авторизация к данной организации
 FnsParticipantId | Строка | Чтение | Идентификатор организации-участника документооборота
 FnsRegistrationDate | Дата | Чтение | Дата подачи заявляения в ФНС на регистрацию данной организации в качестве участника документооборота ЭСФ. В случае, если организация не подавала заявления, возвращается пустая дата
 IsTest | Булево | Чтение | Признак того, что организация работает в тестовом режиме
 IsPilot | Булево | Чтение | Признак того, что организация работает в пилотном режиме

###Методы объекта
 Метод | Описание
 -------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------
 [GetCounteragentListByStatus](1S_GetCounteragentListByStatus) | Возвращает контрагентов, с которыми у организации установлены отношения
 [GetCounteragentListByStatusAsync](1S_GetCounteragentListByStatusAsync) | Асинхронный запрос контрагентов, с которыми у организации установлены отношения
 [GetCounteragentById](1S_GetCounteragentById) | Возвращает контрагента по его идентификатору
 [GetCounteragentListByInnKpp](1S_GetCounteragentListByInnKpp) | Возвращает список контрагентов по ИНН/КПП
 [GetCounteragentListByInnList](1S_GetCounteragentListByInnList) | Инициирует асинхронную операцию получения списка контрагентов по списку ИНН
 [GetDocumentById](1S_GetDocumentById) | Возвращает документ в ящике по его идентификатору
 [GetDocumentsTask](1S_GetDocumentsTask) | Возвращает объект, который позволяет позволяет получить список документов, отправленных из выбранного ящика или пришедших в него
 [CreateSendTask](1S_CreateSendTask) | Позволяет получить задание для отправки документа
 [CreateSendTaskFromFile](1S_CreateSendTaskFromFile) | Позволяет получить задание для отправки документа, загруженного из файла
 [GetUsers](1S_GetUsers) | Вовзращает список всех пользователей организации
 [GetUserPermissions](1S_GetUserPermissions) | Возвращает описание прав пользователя, в контексте которого произошла авторизация, для данной организации
 [GetDocumentEventList](1S_GetDocumentEventList) | Возвращает список событий
 [GetReceiptGenerationProcess](1S_GetReceiptGenerationProcess) | Возвращает объект, с помощью которого можно запустить процесс автоматического формирования документов по регламентному документообороту счетов-фактур
 [AddCertToFnsRegistrationMessage](1S_AddCertToFnsRegistrationMessage) | Добавление в сообщение ФНС нового сертификата
